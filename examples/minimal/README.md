# Minimal Examples

These files are generated by:

```bash
python scripts/make_minimal_examples.py
```

Included outputs:

- `policy_matrix.drawio`: editable diagrams.net source.
- `policy_matrix.pptx`: editable PowerPoint slide.

The example is intentionally lightweight. It demonstrates native text boxes, rounded chips, dashed shift-policy styling, and validation-friendly output formats.
